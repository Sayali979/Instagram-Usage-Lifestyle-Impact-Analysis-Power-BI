# 📊 Instagram Usage & Lifestyle Impact Analysis

An interactive **Power BI Dashboard** that analyzes Instagram usage patterns and their impact on users' lifestyle, health, and engagement behavior.

---

## 📌 Project Overview

Social media has become an essential part of our daily lives. While Instagram provides entertainment and connectivity, excessive usage may affect users' lifestyle, including sleep quality, stress levels, happiness, and engagement.

This Power BI project explores user behavior using interactive dashboards to identify meaningful relationships between Instagram usage patterns and lifestyle indicators.

---

## 🎯 Objectives

- Analyze Instagram usage behavior across different demographics.
- Understand the relationship between Instagram usage and lifestyle factors.
- Compare engagement across different content preferences.
- Analyze audience distribution by country, gender, and employment status.
- Provide interactive dashboards for data-driven insights.

---

# 📈 Key Insights

- Average Instagram usage is **188 minutes per day**.
- Average session duration is **21 minutes**.
- Users sleep approximately **7 hours** on average.
- Full-time employees represent the largest user segment.
- Users aged **18–24** spend the most time on Instagram.
- Live content receives the highest engagement score.
- Heavy Instagram users tend to experience higher stress levels.
- User engagement varies across different content preferences.
